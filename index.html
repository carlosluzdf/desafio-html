INTRODUÇÃO AO HTML

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão utilizada para criar e estruturar conteúdo na web. Ele permite que desenvolvedores de sites definam elementos como texto, imagens, links, listas, tabelas e outros componentes através de uma série de tags.

Esta formação em HTML Web Developer da DIO.me, visa intriduzir o aluno no mundo do Front-end, e é perfeita para começar a jornada entendendo como trabalhar com HTML, a base para criar as primeiras páginas WEB, com oportunidade de aprender na prática a como estruturar a marcação de uma página e como funciona cada tag, conhecendo os elementos semânticos e outras boas práticas na hora de construir a primeira página.

Cliente/Servidor

Representam as duas partes envolvidas em uma operação de navegação, conforme descrito abaixo:

  1. Cliente

    2. Representa o usuário final, acessando um site ou aplicação web, via algum dispositivo, que pode ser um desktop, notebook, smarthphone, etc., e é responsável por realizar as requisições de acordo com sua necessidade.

  2. Servidor

    3. Representa o o lado do processamento das solicitações dos usuários, seguido por respostas em forma de conteúdo HTML, podendo ser necessário a realização de consulta a bancos de dados, entre outras operações que são transparentes para o usuário final.

Tags HTML

Uma tag HTML é um bloco de construção fundamental da linguagem de marcação HTML (HyperText Markup Language), que indica ao navegador como formatar o texto e estruturar o conteúdo de uma página da web.

As tags de conteúdo precisam sempre serem inicadas e finalizadas, com a própria tag precedida do caractere "/". Já as tags de componentes, podem ser finalizadas apenas com o incremento do caractere "/" ao final da própria tab.
